Redis restudy 3 List Set Hash
2023-02-11|字数总计: 2.2k|阅读时长: 11 分钟

List

列表。

应用场景

栈，队列，阻塞队列。

消息队列。

命令

lpush, rpush, lrange

lpush key value [value ...]：向key列表的头部插入值；

rpush key value [value ...]：向key列表的尾部插入值

lrange key a b：查询key列表a-b的值，

lpop, rpop

lpop key：移除列表头部的元素

rpop key：移除列表尾部的元素

lindex, llen

lindex key index：获取key中index位置的值。

llen key：获取列表的长度。

lrem

lrem key count value ：移除指定数量的value值。精确匹配。

ltrim key start stop：裁剪列表，保留start-stop中的元素；

rpoplpush

rpoplpush source destination：从列表source的尾部取出元素，插入到列表destination列表的尾部；

lset

lset key index value ：设置index位置的值为value

  • 只能设置key存在的列表，若key不存在则报错
    • (error) ERR no such key
  • 不返回原值。

linsert

linsert key BEFORE|AFTER pivot element

  • 向值为pivot的位置的前面或后面插入值element。

其他

  1. 本质为一个“链表”
  2. 在两边插入或改动值，效率最高！

Set

集合

set中的值不能重复。

sadd, smembers, sismember, scard

sadd key value [value ...]：向集合添加元素

smembers key：查看元素成员列表

sismember key value：查询value是否为key集合的成员

scard key：获取集合中成员数量

srem

srem key value：移除指定值

srandmember key count：随机取出集合中元素，count为数量，count默认为1

spop

spop key [count]：随机弹出元素，count为数量

smove

smove key1 key2 value：将一个集合中指定的值移动到另一个集合中。

sdiff, sinter, sunion

sdiff set1 set2：获得set1与set2的差集

sinter s1 s2：查询s1和s2的交集

sunion s1 s2：查询s1和s2的并集

Hash（哈希）

key-<key-value> key-Map

值为map集合；

hset, hget, hmget, hmset, hgetall

hset key field value：设置值

hget key field：获取值

hmset key field value [field value ...]：设置多个值

hmget key field [field...]：获取多个值

hgetall key：获取所有值

hdel, hlen

hdel key field [field ...]：删除指定key的field，对应value同时会消失

hlen key：获取hash长度

hexists key field：判断key的field是否存在

hkeys key：获取所有field

hvals key：获取所有values

hincrby, hsetnx

hincrby key field increment：执行自增，可以指定增量

hsetnx key field value：与set一致，不同的是仅用于“key值不存在”时，若存在则不进行
